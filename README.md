# Matcha
<img width="800" alt="" src="https://github.com/user-attachments/assets/772094a4-4dac-436d-ab2a-bae6352e85e7" />

Matcha is a dating app created by [imontero](https://github.com/tentaclepurple) and [ngasco](https://github.com/nicolasgasco) at the [42urdliz](https://www.42urduliz.com/) programming campus.

It is a Full-Stack application written in Python/Flask for the Back-End and JavaScript/SvelteKit for the Front-End.
The application is fully containerized with Docker.

The features of Matcha include:
- Signup flow with confirmation email and password recovery
- Geolocalization and visualization on interactive map ([MapLibre](https://maplibre.org/))
- Filtering and sorting logic when searching for users
- Real-time chat and notification system
- AI-powered users that interact in the Chat
- Mobile styles
